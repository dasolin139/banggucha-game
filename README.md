# 방구차 (Banggucha) 🎮

클래식 한국 레트로 스타일 미로 탈출 게임

## 게임 설명

미로 속에서 방구차(탱크)를 조종하여 모든 깃발을 수집하고 출구로 탈출하세요!

## 조작법

- **방향키** 또는 **WASD**: 이동
- 모든 깃발을 모으면 출구가 열립니다
- 적 탱크를 피하세요!

## 게임 규칙

1. 미로를 탐험하며 빨간 깃발을 모두 수집
2. 모든 깃발을 모으면 초록색 출구가 활성화
3. 출구에 도달하면 스테이지 클리어
4. 적 탱크에 닿으면 라이프 감소 (3개)
5. 스테이지가 올라갈수록 깃발과 적이 증가

## 실행 방법

`index.html`을 브라우저에서 열면 바로 플레이 가능합니다.

## 기술 스택

- HTML5 Canvas
- Vanilla JavaScript
- CSS3 (Press Start 2P 폰트)
